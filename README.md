## KidsStoryApp
<img src="https://user-images.githubusercontent.com/73097560/115834477-dbab4500-a447-11eb-908a-139a6edaec5c.gif"><br><br>
<p align="center">

Esta aplicación es una herramienta educativa diseñada para niños, cuyo propósito es estimular la creatividad y el gusto por la lectura. A través de un sistema de generación de historias, la app crea cuentos personalizados basados en las elecciones del niño, permitiéndole explorar mundos imaginarios, personajes únicos y aventuras divertidas.
El objetivo principal es fomentar la imaginación, la comprensión lectora y el aprendizaje interactivo, ofreciendo una alternativa segura, lúdica y didáctica para el desarrollo infantil. La aplicación no reemplaza la lectura tradicional, sino que la complementa al brindar una experiencia creativa y accesible para los más pequeños.
